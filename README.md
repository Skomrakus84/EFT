# EFT po toksycznym związku - Program 21-dniowy

Ten projekt to interaktywny ebook/strona internetowa stworzona jako program wsparcia dla osób wychodzących z toksycznych relacji.

## Uruchomienie lokalne

Otwórz plik `index.html` w dowolnej przeglądarce internetowej.

## Struktura projektu

- `index.html` - Główny plik strony
- `audio/` - Pliki dźwiękowe z sesjami i powitaniem
- `images/` - Grafiki i karty pracy
- `karta_A.html`, `karta_B.html`, `karta_C.html` - Dodatkowe karty pracy
